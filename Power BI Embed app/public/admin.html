ℹ️ Scope of access
Only users and guests of this Microsoft 365 tenant are able to login to this portal. The rules below apply within that boundary.

Sign-in control

Allow new sign-ins
If off, only users already in the Users list can log in. New accounts will be denied.
How rules work Allow domain/email — if any allow rules exist, only listed users or domains can log in.
Block domain/email — always applied, even if the user matches an allow rule.
If no rules are configured, any user who can authenticate with this Microsoft 365 tenant can access the portal.

Portal branding

Customise the name, colours, logo and footer. Changes take effect on page refresh.

Colours

Accent — buttons, active states, highlights, the connected indicator.
Navigation background — the top nav bar, sub-bar, and report sidebar.
Page text — headings and body text on white page areas. Darken if you lighten the page background.
Muted text — labels, table headers, secondary information. Should be lighter than page text.
